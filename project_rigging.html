Mechanical Rigging

3ds Max

Return

For this project I modeled a simple version of a concept from Shimmering-Sword so I could have a first go at mechanical rigging. All joints are individually controllable and the arms use an IK/FK blend with snapping.

Challenges

  • Using MaxScript to make IK/FK Snapping
  • Importing a skinned mesh into Unreal Engine
  • Learning basic animation in 3ds max

Technology Used

  • Unreal Engine
  • Autodesk 3ds Max
