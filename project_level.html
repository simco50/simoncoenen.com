DM-Summit

Unreal Tournament

Return

The goal of this was to create a modular level, suited for 2-4 players, using the Unreal Tournament Editor. The mesh work is done in 3ds max and everything is textures using the ‘Ultimate Trim’ method (crf. Sunset Overdrive at GDC). I’ve had tons of fun working on this even though I’m engaged more in programming. I definitely learned a lot from this project.

Challenges

  • Learning the Unreal Engine workflow
  • Creating assets that are modular
  • Making shaders using the node system in Unreal Engine
  • Texturing using the ‘Ultimate Trim’ method (PBR)
  • Working at a high pace while still having a good result

Technology Used

  • Unreal Engine
  • Autodesk 3ds Max
  • Adobe Photoshop
  • xNormal
