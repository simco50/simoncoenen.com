A.N.N.E.

Game made in 2D Engine with Box2D

Return

This game is inspired by A.N.N.E. which is an upcoming sidescroller platform. I tried to recreate a particular part of the game using sprites from OpenGameArt The game is made using an existing 2D game engine with Box2D integration.

Challenges

  • Getting to know object oriented programming
  • Usage of spritesheets
  • Vector image based collition

Technology Used
